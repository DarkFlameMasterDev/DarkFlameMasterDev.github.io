[考研算法] 001-循环左移

所有的考研题目，为了考试的统一性和易学性，均采用C语言来编写，版本为C17，使用其他版本也可以，仅仅停留在算法层面，各大版本几乎区别不大

【2010年统考真题】设将n(n>1)n(n>1)n(n>1)个整数存放到一维数组RRR中。设计一个在时间和空间两方面都尽可能高效的算法。将RRR中保存的序列循环左移p(0<p<n)p(0<p<n)p(0<p<n)个位置，即将RRR中的数据由(X0,X1,...,Xn−1)(X_0,X_1,...,X_{n-1})(X0​,X1​,...,Xn−1​)变换为(Xp,Xp+1,...,Xn−1,X0,X1,...,Xp−1)(X_p,X_{p+1},...,X_{n-1},X_0,X_1,...,X_{p-1})(Xp​,Xp+1​,...,Xn−1​,X0​,X1​,...,Xp−1​)。

三次逆置法

这是一个经典的算法问题。
为了在时间和空间两方面都尽可能高效，最佳的方法是采用 “三次逆置法”（Reverse Algorithm）。

该算法可以在 O(n)\displaystyle O(n)O(n) 的时间复杂度和 O(1)\displaystyle O(1)O(1) 的空间复杂度下完成任务。

1. 算法设计思想

我们可以将数组 RRR 视为由两个部分组成的：

  • 前部 AAA：(X0,X1,...,Xp−1)\displaystyle (X_0, X_1, ..., X_{p-1})(X0​,X1​,...,Xp−1​)
  • 后部 BBB：(Xp,Xp+1,...,Xn−1)\displaystyle (X_p, X_{p+1}, ..., X_{n-1})(Xp​,Xp+1​,...,Xn−1​)

原始数组为 ABABAB，我们需要将其转换为 BABABA。
利用向量逆置的性质 (ATBT)T=BA\displaystyle (A^T B^T)^T = BA(ATBT)T=BA，我们可以通过以下三个步骤实现：

  1. 逆置前部：将数组的前 p\displaystyle pp 个元素 (X0,...,Xp−1)\displaystyle (X_0, ..., X_{p-1})(X0​,...,Xp−1​) 原地逆置。
  2. 逆置后部：将数组的剩余 n−p\displaystyle n-pn−p 个元素 (Xp,...,Xn−1)\displaystyle (X_p, ..., X_{n-1})(Xp​,...,Xn−1​) 原地逆置。
  3. 整体逆置：将整个数组 (X0,...,Xn−1)\displaystyle (X_0, ..., X_{n-1})(X0​,...,Xn−1​) 原地逆置。

经过这三步，数组中的数据就变成了 (Xp,...,Xn−1,X0,...,Xp−1)\displaystyle (X_p, ..., X_{n-1}, X_0, ..., X_{p-1})(Xp​,...,Xn−1​,X0​,...,Xp−1​)，即完成了循环左移。

3. 复杂度分析

  • 时间复杂度：
    该算法主要由三次逆置操作组成。

    • 第一次逆置涉及 p\displaystyle pp 个元素，操作次数约为 p/2\displaystyle p/2p/2。
    • 第二次逆置涉及 n−p\displaystyle n-pn−p 个元素，操作次数约为 (n−p)/2\displaystyle (n-p)/2(n−p)/2。
    • 第三次逆置涉及 n\displaystyle nn 个元素，操作次数约为 n/2\displaystyle n/2n/2。
    • 总的操作次数为 2n\displaystyle 2n2n 次扫描。
      因此，时间复杂度为 O(n)\displaystyle O(n)O(n)。

  • 空间复杂度：
    算法是在原数组上进行操作（In-place），只需要用到有限的几个辅助变量（如 i, temp, from, to）来进行交换和循环控制，不需要额外的数组空间。
    因此，空间复杂度为 O(1)\displaystyle O(1)O(1)。

4. 示例推演

假设 R=(0,1,2,3,4)\displaystyle R = (0, 1, 2, 3, 4)R=(0,1,2,3,4)，n=5\displaystyle n=5n=5，p=2\displaystyle p=2p=2。目标结果为 (2,3,4,0,1)\displaystyle (2, 3, 4, 0, 1)(2,3,4,0,1)。

  1. 逆置前 p\displaystyle pp 个 (0\displaystyle 00 到 1\displaystyle 11)：
    (1,0,2,3,4)\displaystyle (1, 0, 2, 3, 4)(1,0,2,3,4)
  2. 逆置后 n−p\displaystyle n-pn−p 个 (2\displaystyle 22 到 4\displaystyle 44)：
    (1,0,4,3,2)\displaystyle (1, 0, 4, 3, 2)(1,0,4,3,2)
  3. 整体逆置 (0\displaystyle 00 到 4\displaystyle 44)：
    (2,3,4,0,1)\displaystyle (2, 3, 4, 0, 1)(2,3,4,0,1)
